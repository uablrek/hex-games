Basic Map

The Map is draggable by grab-and-drag, but not scrollable.

  • Place a marker. Shows coordinates
  • Drag pre-placed counters. They will snap to hex

Show Counters (without map)

The counter sheets are converted to PNG. Individual counters are cut from these sheets. The index is very central! Counters are identified by their's index. The counter images have id=`rdtr${i}`. Counters are draggable

  • Counters by index
  • Counters by nation

Basic Save/Restore

Save is complicated since scripts may not write to the local file system. But a browser download can be initiated. Restore is done from the json file "test-deployment.json" which must be in the app directory.

  • Deploy Netrals from a draggable box. Save with Shift-S
  • Restore deployment from "test-deployment.json"

Manually play the 1939/Campaign scenario

No rules enforced nor help with book-keeping (e.g BRP). You must do everything manually, including compute odds and rolling the die, just like the board game. Loads from "rdtrSaveData.js" if present in the app directory. Save with Shift-S, and copy
