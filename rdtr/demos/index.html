Manually play a scenario

No rules enforced nor help with book-keeping (e.g BRP). You must do almost everything manually, like compute odds remove causalities. Loads from "rdtrSaveData.js" if present in the app directory. Save with Shift-S. Copy the downloaded save to "rdtrSaveData.js" in the app directory, and reload the page (F5) to restore.

Demos

For developers and curious people

Map

The Map is draggable by grab-and-drag, but not scrollable.

  • Place a marker. Shows coordinates
  • Drag pre-placed counters. They will snap to hex
  • Test movement algorithm

Show Counters (without map)

The counter sheets are converted to PNG. Individual counters are cut from these sheets. The index is very central! Counters are identified by their's index. The counter images have id="rdtr#", where "#" is the index. Counters are draggable

  • Counters by index
  • Counters by nation
  • Generated counters

Basic Save/Restore

Save is complicated since scripts may not write to the local file system. Instead a browser download is initiated. Restore is done from the json file "test-deployment.json" which is included at build time (please se comments in "restore-demo.js").

  • Deploy Netrals from a draggable box. Save with "s"
  • Restore deployment from "test-deployment.json"
